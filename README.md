# FinalProjectCMPSC132

This is a simple number guessing game. The computer generates a random number 1-100 and the user, you, have 10 attempts to guess the number. The program will give feedback based on your input. 

Types of feedbacks
- Too high: try choosing a lower number
- Too low: try choosing a lower number
- Congratulations!: you won the game
- Out of Bounds: please choose a number 1-100
- INVALID!!!: please choose a number

You will start off with the plot of the game. Click next to continue. When the plot is over, click the start button to start the actual game. A textbox will show. It is where you can type your input. Click the guess button to submit input. After you either won the game or ran out of attempts, a new button, go back in time, will show. Click to restart the game.

Note:
If the same feedback is gotten twice, nothing will change on the UI but your input is submitted.
